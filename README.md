Музыкальный Discord-бот на Python с удобными slash-командами. 

## Возможности
-  Воспроизведение музыки с YouTube через `yt-dlp` + `FFmpeg`
-  Очередь треков для каждого сервера (guild’а)
-  Режимы:
  - `/loop` — зацикливание всей очереди  
  - `/repeat` — повтор текущего трека
-  Управление:
  - `/skip` — пропустить трек  
  - `/queue` — посмотреть текущую очередь  
  - `/volume` — изменить громкость (0–100%, по умолчанию 5%)  
  - `/bass` — изменить уровень баса (от -10 до +20 dB через фильтр FFmpeg)
-  История прослушиваний:
  - `/history` — показывает последние 15 треков, сохранённых в базе
-  Умный поиск:
  - `/play` — поиск по YouTube с выводом выборки в виде Embed + кнопки выбора трека
-  Логирование:
  - лог-файл с ротацией (`RotatingFileHandler`)
  - вывод логов в консоль
-  Хранение данных:
  - PostgreSQL + async SQLAlchemy (`asyncpg`)
  - отдельная таблица `TrackHistory` с информацией о проигранных треках
